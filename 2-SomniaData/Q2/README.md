# Q2 - SQL Schema (4 points)

**Objective:** Using your Entity-Relationship Diagram, write the SQL schema in `SCHEMA.sql`.

## Instructions

- Modify the `SCHEMA.sql` file in this directory
- Use the MySQL language if possible
- Implement all tables identified in your Entity-Relationship diagram
- Define appropriate primary and foreign keys
- Respect naming best practices

## Expected elements in the SQL schema

### Tables

- Table of main entities
- Linking tables if necessary

### Keys

- Primary keys for each table
- Foreign keys for relationships between tables

### Data types

- Appropriate choice of data types
- Consistent field sizes
- Management of nullable values
