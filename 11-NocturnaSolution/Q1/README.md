# Q1 - Logology (7 points)

**Objective:** Design a script that generates an ASCII art logo for "Nocturna Solutions" algorithmically.

## Context

Nocturna Solutions wants to modernize its image and adopt an original logo. However, due to budget constraints, the marketing team decided that this logo must be generated in ASCII art directly from code rather than created by a graphic designer.

## Instructions

- Write a program (in the language of your choice) that draws the logo in the console (standard output), line by line or character by character, using programming logic.
- The logo must represent the company "Nocturna Solutions" with visual elements evoking themes such as night, sleep, etc.
